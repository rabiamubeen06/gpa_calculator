# 🎓 GPA Calculator

A clean, interactive **GPA & CGPA calculator** built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies, just pure web.

🔗 **Live Demo:** [student-gpa-calc.netlify.app](https://student-gpa-calc.netlify.app/)

---

## ✨ Features

- 📚 **Multi-semester support** — add as many semesters as needed
- 🧮 **Per-semester GPA calculation** with credit-hour weighting
- 🔄 **Live CGPA tracking** across all semesters
- ➕ **Dynamic course rows** — add or remove courses on the fly
- 🎯 **Grade point display** updates instantly on grade selection
- 💡 **Visual GPA highlight** animation on calculation
- 📱 **Responsive design** — works on desktop and mobile

---

## 🚀 Getting Started

No build tools or installations required.

```bash
git clone https://github.com/your-username/gpa-calculator.git
cd gpa-calculator
```

Then open `index.html` in your browser — that's it.
